
# Stock Market Prediction using Machine Learning

## Overview
This project predicts stock prices using multiple ML models like:
- LSTM
- KNN
- Linear Regression

## Features
- Data preprocessing
- Model comparison
- Visualization
- Flask-based web interface

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- TensorFlow/Keras
- Flask

## How to Run
1. Install dependencies
2. Run app.py
3. Open localhost

## Note
This is a learning-based project.
